# Assistance

L'aide est organisée en deux volets, selon le côté de Hecate où vous vous
trouvez :

<div class="grid cards" markdown>

-   :material-account-hard-hat: __Opérateurs__ · Saisie & Viewer

    ---

    Vous utilisez l'application de saisie ou Hecate Viewer sur le terrain —
    scanner, localiser, se connecter à un broker, regarder les actifs arriver
    sur la carte en direct.

    [:octicons-arrow-right-24: Assistance opérateurs](operator/index.md)

-   :material-tune-variant: __Administrateurs__ · Application admin

    ---

    Rédiger des profils, configurer le broker, publier et retirer des
    profils, gérer les identifiants.

    [:octicons-arrow-right-24: Assistance admin](admin/index.md)

</div>
