# Support

Die Hilfe ist in zwei Spuren aufgeteilt, je nachdem, auf welcher Seite von
Hecate Sie stehen:

<div class="grid cards" markdown>

-   :material-account-hard-hat: __Bediener__ · Erfassung & Viewer

    ---

    Sie nutzen die Erfassungs-App oder Hecate Viewer im Feld — scannen,
    verorten, mit einem Broker verbinden, Assets live auf der Karte
    eintreffen sehen.

    [:octicons-arrow-right-24: Bediener-Support](operator/index.md)

-   :material-tune-variant: __Administratoren__ · Admin-App

    ---

    Profile verfassen, den Broker einrichten, Profile veröffentlichen und
    zurückziehen, Zugangsdaten verwalten.

    [:octicons-arrow-right-24: Admin-Support](admin/index.md)

</div>

## Kontakt

**iOS-Version**, die **App-Version** sowie an, was Sie getan und was Sie
