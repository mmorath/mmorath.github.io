# Support

Help is split into two tracks, depending on which side of Hecate you're on:

<div class="grid cards" markdown>

-   :material-account-hard-hat: __Operators__ · Capture & Viewer

    ---

    Using the iPhone/iPad capture app or the Apple TV viewer in the field —
    scanning, locating, connecting to a broker, watching the live wall.

    [:octicons-arrow-right-24: Operator support](operator/index.md)

-   :material-tune-variant: __Administrators__ · Admin app

    ---

    Authoring profiles, setting up the broker, publishing and retiring profiles,
    managing credentials.

    [:octicons-arrow-right-24: Admin support](admin/index.md)

</div>
